Cloud Storage Tiers

When using cloud storage the tier offer different levels or security and features.
If you choose a local file the file will be pulled directly from the app itself (Installed Native Apps). PWAs automatically use cloud storage<

Tier 1

  • Least Security
  • No Analytics
  • No Access Control
  • Fast Loading

Tier 2

  • Average Security
  • Analytics Possible
  • Database Control
  • Faster Loading

Tier 3

  • Industry Standard Security
  • Analytics
  • Customized Access Control Lists
  • Fastest Loading
  • Admin Panel
